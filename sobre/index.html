Acerca del sitio


Descripción

Este sitio tiene como propósito ser un portfolio simple sobre mis trabajos gráficos tanto personales como los realizados en la universidad. Hay distintas secciones donde el lector puede ver una galería en el apatado "descripción", puede saber sobre mí en su respectivo apartado, y también puede comunicarse conmigo a través de la sección "formulario.



Desarrollo

El desarrollo fue realizado a partir de HTML, CSS y Javascript, pero principalmente a través de CSS. Los códigos CSS usados para la tarjeta de presentación, la galería y el formulario fueron prestados de fuentes libre de uso para uso educativo. Estos códigos los fui modificando para ajustar las necesidades del sitio. También cabe destacar el uso de cajas contenedoras para los textos y animaciones que reaccionan al comando "hover", de modo para brindarle más interés al diseño. Fue fundamental para poder estilizar adecuadamente cada sección el uso de <div></div> con clases personalizadas.

Para la validación de e-mail, utilicé patrones en HTML para que el navegador sólo permita proseguir cuando aparezcan determinados caracteres en el campo de texto.

Conclusiones

Aunque exhaustivo al final, la realización de este proyecto fue una tarea muy enriquecedora y aprendí mucho investigando. Me encontré durante el proceso con muchos obstáculos para poder generar el estilo que yo quería que el sitio tuviese pero creo que el resultado fue satisfactorio. Creo que de todas las cosas que aprendí de HTML, CSS y Javascript, lo que definitivamente me ha ayudado muchísimo fue utilizar las secciones <div></div>, recuerdo que al comienzo de la generación del sitio me resultaba difícil entender el concepto hasta haberlo puesto propiamente en práctica muchas veces.
